Hướng dẫn thêm game lên GitHub Pages

  1. Đặt file game HTML vào thư mục games/tên-game/index.html trong repo.
  2. Điền thông tin ở form trên → nhấn Thêm vào danh sách.
  3. Nhấn Tải file để tải games.json mới.
  4. Thay thế games.json cũ trong repo bằng file vừa tải.
  5. Commit & push lên GitHub — trang chủ sẽ tự cập nhật.
